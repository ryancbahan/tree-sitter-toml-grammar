===================
Whitespace in Table Headers
===================

[a.b.c]            # this is best practice
[ d.e.f ]          # same as [d.e.f]
[ g . h . i ]      # same as [g.h.i]
[ j . "Ê" . 'l' ]  # same as [j."Ê".'l']
